## 研究摘要

本研究探討 PTT 股票板的情緒與台股大盤走勢之間的關聯性，透過：

1. 自動化爬蟲收集 PTT Stock 板文章與推文
2. BERT 情緒分類進行三分類（正面/中性/負面）
3. 人工標註系統進行驗證
4. 統計檢定分析（卡方檢定、Spearman 相關性、Bootstrap）
